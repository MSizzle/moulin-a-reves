# Plan 06-03 — Status canary script

## Goal

Add a dual-mode canary that submits a small dry-run batch through `/api/feedback/submit`, then polls `/api/feedback/status/<N>` until it stabilizes at stage 5 OR a terminal review sub-state. Mirrors the shape of `scripts/smoke-feedback-v2.mjs` — unit mode (with TARGET_URL unset) for in-build regression, canary mode (with TARGET_URL=https://www.moulinareves.com) for post-deploy verification.

## Files

| `scripts/smoke-feedback-status.mjs` | NEW — canary script |
| `scripts/canary.sh` | EDIT — add `status` dispatch |
| `package.json` | EDIT — add `canary:status` script; extend `canary` (no-arg) to run v1→v2→status sequentially |

## Tasks

### Task 1 — `scripts/smoke-feedback-status.mjs`

Structure:

```js
#!/usr/bin/env node
// Status-endpoint canary.
//
// UNIT MODE (no TARGET_URL): imports resolveStage from src/lib/feedback-status.ts
// directly and runs ~6 table-driven assertions covering stages 1..5 + the
// missing-Vercel-token degrade path. Exits non-zero on any mismatch.
//
// CANARY MODE (TARGET_URL set): mints a maison_session via createSession() from
// src/lib/auth.ts, posts a DRY_RUN v2 batch with 1 trivial edit, then polls
// /api/feedback/status/<N> every 8 s up to 5 minutes. Pass = reached stage 5
// OR sub in {'needs-review','needs-client-reply'}. Fail = timeout, regression,
// or HTTP error.

const TARGET_URL = process.env.TARGET_URL || null;
process.env.DASHBOARD_PASSWORD ||= 'moulin2024';

if (!TARGET_URL) {
  // Unit-mode table tests
  const { resolveStage } = await import('../src/lib/feedback-status.ts');
  const cases = [
    {
      name: 'stage 1 — fresh issue no label',
      input: { issueLabels: [], issueHasPrLink: false, prFromComments: null, pr: null, vercelDeployState: null },
      expect: { stage: 1, sub: null },
    },
    {
      name: 'stage 2 — labelled, no PR',
      input: { issueLabels: ['client-feedback'], issueHasPrLink: false, prFromComments: null, pr: null, vercelDeployState: null },
      expect: { stage: 2, sub: null },
    },
    {
      name: 'stage 3 — PR open, no terminal label',
      input: { issueLabels: ['client-feedback'], issueHasPrLink: true, prFromComments: null,
        pr: { state: 'open', merged: false, mergeSha: null, url: 'x', number: 1 }, vercelDeployState: null },
      expect: { stage: 3, sub: 'open' },
    },
    {
      name: 'stage 4 needs-review',
      input: { issueLabels: ['client-feedback','needs-review'], issueHasPrLink: true, prFromComments: null,
        pr: { state: 'open', merged: false, mergeSha: null, url: 'x', number: 1 }, vercelDeployState: null },
      expect: { stage: 4, sub: 'needs-review' },
    },
    {
      name: 'stage 4 auto-merged, no vercel token',
      input: { issueLabels: ['auto-merged'], issueHasPrLink: true, prFromComments: null,
        pr: { state: 'closed', merged: true, mergeSha: 'abc', url: 'x', number: 1 }, vercelDeployState: null },
      expect: { stage: 4, sub: 'auto-merged' },
    },
    {
      name: 'stage 5 — merged + vercel READY',
      input: { issueLabels: ['auto-merged'], issueHasPrLink: true, prFromComments: null,
        pr: { state: 'closed', merged: true, mergeSha: 'abc', url: 'x', number: 1 }, vercelDeployState: 'READY' },
      expect: { stage: 5, sub: null },
    },
  ];
  let pass = 0, fail = 0;
  for (const c of cases) {
    const got = resolveStage(c.input);
    const ok = got.stage === c.expect.stage && got.sub === c.expect.sub;
    console.log((ok ? 'PASS' : 'FAIL') + ' — ' + c.name + ' (got stage=' + got.stage + ' sub=' + got.sub + ')');
    ok ? pass++ : fail++;
  }
  console.log('\n' + pass + '/' + cases.length + ' passed');
  process.exit(fail ? 1 : 0);
} else {
  // Canary mode: real fetch against TARGET_URL
  process.env.GITHUB_TOKEN ||= 'canary-noop'; // not used in canary mode
  process.env.GITHUB_REPO ||= 'MSizzle/moulin-a-reves';
  const { createSession } = await import('../src/lib/auth.ts');
  const cookie = 'maison_session=' + encodeURIComponent(await createSession());

  // Submit a trivial v1 single-edit (cheaper than v2 batch; same status pipeline)
  const submitPayload = {
    schemaVersion: 1,
    intent: 'change-wording',
    locator: { schemaVersion: 1, pageRoute: '/', astroFileGuess: 'src/pages/index.astro',
      i18nKey: 'nav.homes', i18nAttr: 'data-i18n', nearbyText: 'Houses', _currentText: 'Houses' },
    intentDetail: { newTextEn: 'Houses' /* no-op text — Action should skip or auto-close */, newTextFr: 'Maisons', okToTranslate: true },
    isTest: true,
  };
  const startRes = await fetch(TARGET_URL + '/api/feedback/submit', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json', cookie },
    body: JSON.stringify(submitPayload),
  });
  if (!startRes.ok) {
    console.error('FAIL submit:', startRes.status, await startRes.text());
    process.exit(1);
  }
  const startData = await startRes.json();
  if (!startData.ok || !startData.issueNumber) {
    console.error('FAIL submit shape:', startData);
    process.exit(1);
  }
  const num = startData.issueNumber;
  console.log('Submitted issue #' + num + '; polling status…');

  const deadline = Date.now() + 5 * 60 * 1000;
  let lastStage = 0, lastSub = null;
  while (Date.now() < deadline) {
    await new Promise(r => setTimeout(r, 8000));
    const sRes = await fetch(TARGET_URL + '/api/feedback/status/' + num, { headers: { cookie } });
    if (!sRes.ok) { console.error('status http', sRes.status); continue; }
    const s = await sRes.json();
    if (s.stage < lastStage) {
      console.error('FAIL regression — stage went from', lastStage, 'to', s.stage);
      process.exit(1);
    }
    lastStage = s.stage; lastSub = s.sub;
    console.log('  stage=' + s.stage + ' sub=' + s.sub);
    if (s.stage === 5 || s.sub === 'needs-review' || s.sub === 'needs-client-reply') {
      console.log('PASS — terminal state reached');
      process.exit(0);
    }
  }
  console.error('FAIL timeout — last stage=' + lastStage + ' sub=' + lastSub);
  process.exit(1);
}
```

### Task 2 — `scripts/canary.sh` dispatch + `package.json`

Add `status` to canary.sh's switch:

```sh
case "${1:-}" in
  v1) ... ;;
  v2) ... ;;
  status) exec npx tsx scripts/smoke-feedback-status.mjs ;;
  "") npx tsx scripts/smoke-feedback-v2.mjs --canary v1 && \
      npx tsx scripts/smoke-feedback-v2.mjs --canary v2 && \
      npx tsx scripts/smoke-feedback-status.mjs ;;
  *) ... ;;
esac
```

Add to `package.json` scripts:

```json
"canary:status": "scripts/canary.sh status",
```
